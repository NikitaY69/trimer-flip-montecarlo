Detailed Description

Structures and functions for particle configurations and interactions.

This module defines the configuration structure to represent the state of particles in a simulation, including their positions, bonded interactions, and Verlet lists. It also provides utility functions for handling periodic boundary conditions and managing particle coordinates.
